# Windows 服务器使用说明

## 端口说明

**默认端口**: 8888

如果需要修改端口,请编辑 `server.js` 文件第 657 行:
```javascript
const PORT = process.env.PORT || 8888;  // 修改这里的数字
```

## 启动服务

### 方式一: 直接启动
双击运行 `启动服务器.bat`

### 方式二: 重启模式(推荐)
双击运行 `重启服务器.bat` - 会自动停止旧进程后启动新服务

## 停止服务

### 方式一: 在启动窗口按 Ctrl+C
在运行的服务器窗口按 `Ctrl + C` 即可停止

### 方式二: 使用停止脚本
双击运行 `停止服务器.bat` - 会强制停止所有 node.exe 进程

## 端口冲突处理

如果 8888 端口被占用,可以选择:

1. **修改端口号**: 编辑 `server.js` 第 657 行
2. **停止占用端口的程序**:
   - 按 `Win + R`,输入 `cmd` 回车
   - 输入: `netstat -ano | findstr :8888`
   - 找到 PID 后输入: `taskkill /F /PID <PID数字>`

## 访问地址

启动成功后,浏览器访问:
- 本地访问: `http://localhost:8888`
- 局域网访问: `http://[你的IP地址]:8888`

## 注意事项

- 关闭服务器窗口**不会**自动停止服务,必须使用上述停止方法
- 如果程序无响应,使用 `停止服务器.bat` 强制停止
- 修改配置文件后需要重启服务才能生效
